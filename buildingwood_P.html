빌딩숲
[Servlet Project]
직장인들을 위한 다목적 커뮤니티 사이트
1. 기획서
기획서
기획서를 작성하며 주제선정 및 분석을 통해 기존의 문제점 및 개선점을 알아보고 구현목표를 세웁니다.
2. 요구분석서
요구분석서
요구분석서를 통해 기능적 요구사항을 상세하게 서술합니다.
3. 화면설계(Figma)
화면 설계(Figma)
Figma를 활용한 화면 설계를 진행하였습니다.
4. 페이지 설계(Drowio)
페이지 설계(Drowio)
Drowio를 활용하여 페이지 간의 이동과 기능을 시각적으로 표현하였고 패키지와 URL주소를 정하였습니다.
5. ERD
ERD

기능적 요구사항을 충족시키기위한 빌딩숲의 DBMS입니다.

11. 느낀 점
  • 학원에서 진행하는 첫 웹사이트 프로젝트로 할 수 있을까라는 부담감이 심했습니다. 그래서 자습용으로 간단한 웹사이트를 만들었고 이를 통해 자신감을 가지게 되었습니다.
  • 프로젝트 진행 중 코로나 확진으로 신체적으로 많이 고되었지만 책임감을 가지고 성공적으로 끝마쳐 너무 뿌듯했습니다.
  • 처음에는 API 사용하는 것에 익숙지 않아 힘들었지만 후반부가 될수록 사용하는 데에 수월함을 느끼게 되었습니다.
  • 프로젝트를 진행하며 Cookie와 Section의 생명 주기, 보안에 대해 좀 더 이해하게 되었습니다.
  • 처음으로 만든 웹사이트 프로젝트로 너무 재미있었고 개발자의 발전은 무궁무진하다는 것과 더욱 발전에 증진해야겠다고 느꼈습니다.
